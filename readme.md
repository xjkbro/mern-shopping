# RepTracker

MERN Stack application to track and manage workouts.

[RepTracker](https://github.com/xjkbro/RepTracker.git)

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

What things you need to install the application

- git
- npm


### Installing

1. Clone the repo

```
git clone https://github.com/xjkbro/RepTracker.git
```

2. Open Project Folder

3. Install dependancies

```
npm run client-install      //installs client dependancies
npm install                 //installs server dependancies
```

4. Happy Testing! :)
